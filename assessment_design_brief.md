# EUNOIA ASSESSMENT - GAMIFIED UI DESIGN BRIEF

## Overview
Create interactive, gamified UI screens for a mental health assessment with **43 questions** across **6 sections**. Each question should have its own unique, engaging visual experience.

## Design Goals
- **Reduce cognitive load**: One question per screen
- **Visual storytelling**: Use emojis, illustrations, animations that relate to each question
- **Smooth transitions**: Gentle animations between questions
- **Progress clarity**: Show section progress and overall progress
- **Empathetic tone**: Warm, supportive, never clinical

---

## SECTION 1: MOOD (9 questions)
**Theme:** 💙 Heart, emotions, feelings
**Color:** Warm amber (#C17B2F)
**Scale:** PHQ-9 (Depression screening)

2. Feeling down, depressed, or hopeless
3. Trouble falling or staying asleep, or sleeping too much

8. Moving or speaking so slowly that other people could have noticed? Or the opposite — being so fidgety or restless that you have been moving around a lot more than usual

## DESIGN SPECIFICATIONS

### Layout Structure (per question screen):
```
┌────────────────────────────┐
│  Progress Bar (thin)       │ ← Section progress
├────────────────────────────┤
│                            │
│   [Visual Element]         │ ← Emoji/illustration (large)
│                            │
│   "Question text here"     │ ← 18-24px, serif font
│                            │
│   [ ] Option 1            │ ← Radio/tap buttons
│   [ ] Option 2            │   Rounded, soft shadows
│   [ ] Option 3            │
│   [ ] Option 4            │
│                            │
│   Question X of Y          │ ← Small, light gray
└────────────────────────────┘
   [← Back]         [Next →]  ← Navigation (Next auto-advances)
```

### Visual Style:
- **Background:** Soft gradient (warm to cool based on section color)
- **Cards:** Glassmorphic frosted effect (like in Circles)
- **Typography:** 
  - Questions: Serif (Georgia, Lora)
  - Options: Sans-serif (Inter, system)
- **Animations:**
  - Fade in: 300ms ease
  - Slide transition: 400ms ease-out
  - Button press: Scale 0.98
  - Selection: Gentle glow effect

### Interactive Elements:
- **Tap to select:** Option buttons have clear selected state (filled bg, border glow)
- **Auto-advance:** After selection, wait 300ms then slide to next question
- **Progress celebration:** At end of each section, show completion message with animation
- **Safe exit:** "Save & Continue Later" button always visible (top corner)

## CRITICAL: Sensitive Content Handling

**Question phq9_9 (Mood section, Q9):**
_"Thoughts that you would be better off dead, or of hurting yourself in some way"_

**Special Design Requirements:**
- Softer visual (no sharp imagery)
- Gentle support message below: "Your honesty is brave. If you're in crisis, help is available 24/7."
- Crisis hotline button: "Get Help Now" (prominent, warm color)
- Extra padding/breathing room around this question

## OUTPUT REQUEST

**For Each Question (1-43), provide:**

1. **Screen Title:** Question number and section
2. **Main Visual Element:** 
   - Emoji (if simple) OR
   - Illustration description (if complex)
   - Animation notes (if any)
3. **Color Palette:** Primary, secondary, accent colors
4. **Typography:** Font sizes, weights, colors
5. **Layout Notes:** Spacing, alignment, card dimensions
6. **Interaction:** How selection/transition works
7. **Special Effects:** Shadows, glows, parallax, etc.

**Deliverable Format:**
- PDF with one page per question screen (43 pages total)
- Visual mockup or detailed description for each
- Consistent design system across all screens
- Mobile-first (375px - 428px width)

## NOTES FOR AI GENERATION

- **Tone:** Empathetic, supportive, never clinical
- **Accessibility:** High contrast, clear tap targets (min 44px)
- **Performance:** Lightweight animations, quick loads
- **Privacy:** No personally identifiable elements in visuals
- **Cultural sensitivity:** Universal symbols, avoid culture-specific imagery

**End Goal:** Make a 43-question assessment feel like a 5-minute guided meditation, not a medical exam.
